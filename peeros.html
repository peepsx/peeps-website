The Decentralized OS.

221 days 09:19:00

PeerOS is a completely reimagined version of Android for desktop and mobile devices, that utilizes various peer-to-peer technologies and a distributed file system to protect the privacy and rights of its users. PeerOS launches January 1st, 2022.

Read PeerOS Introduction
keyboard_arrow_down

Photograph of Mt. Fuji

universal and secure authentication layer.

PeerOS uses a universal authentication layer that is both cryptographically secure and impossible to hack, instead of the weak forms of local authentication used by traditional operating systems.

keyboard_arrow_down

the file system reimagined.

PeerOS is built on a distributed file system known as dWebFS, where files can only be edited by the owner and where all edits are cryptographically validated through PeerOS's universal authentication layer. This makes it impossible for hackers to inject malware into your file system, without hackers cryptographically validating themselves - and that would be mathematically impossible.

keyboard_arrow_down

powerful decentralized applications.

PeerOS comes fully loaded with many powerful decentralized applications (dApps) like dPhone, dMessenger, dSocial, dRide, dMaps, dBrowser, dWallet, dTunes, dPay and more, ultimately empowering its users to control their own destiny.

keyboard_arrow_down

a massive decentralized ecosystem.

When PeerOS launches this upcoming January, it will feature a powerful platform for developing decentralized applications, as well as a dApp store, birthing a massive decentralized ecosystem never before seen on a personal computer or mobile device.
